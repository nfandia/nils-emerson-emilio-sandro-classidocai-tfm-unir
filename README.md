ClassiDocAI

# Sistema inteligente para la automatización de la gestión documental - CyDoc

Repositorio oficial del Trabajo de Fin de Máster (TFM) de la Universidad Internacional de La Rioja (UNIR).

## Descripción
Este proyecto implementa un sistema basado en Inteligencia Artificial para optimizar la clasificación y búsqueda en la plataforma CyDoc.

## Estructura del repositorio
- `/data`: Datasets utilizados (anonimizados).
- `/src`: Código fuente de los modelos (BERT, Sentence-BERT).
- `/notebooks`: Cuadernos de experimentación (CRISP-DM).
